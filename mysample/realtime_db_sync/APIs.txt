Realtime db sync sample APIs
=======================

## getList

  get the key list
  req
    param: username
    body:  none
  response
    body:
      key1
      key2
      key3
      key6
       .
       .
  
## getValue

  get the value for the key.
  req
    param: username, key
    body:
  response
    body:
      enclosed base64 data.

## insertValue

  insert a new value
  req
    param: username, key ( key must be generated by client )
    body:
      enclosed base64 data.
  response
    body:  none


## deleteValue

  delete a value
  req
    param: username, key
    body:  none
  response
    body:  true/false
